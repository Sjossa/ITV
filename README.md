# ITV
Une application pour aider les introvertis à progresser dans leur sociabilisation en toute confiance.

## Description

ITV (Inner Thrive Vibes) est une application conçue pour aider les introvertis à améliorer leur interaction sociale de manière progressive et respectueuse. L’application propose des missions simples, évolutives, un chat sécurisé, et un contenu psychologique pour accompagner ses utilisateurs dans leur développement personnel. Le tout dans une ambiance douce, sereine et respectueuse de la vie privée.

## Fonctionnalités

- **Missions progressives** : Des défis à relever dans un ordre évolutif pour améliorer la confiance en soi.
- **Chat sécurisé** : Communication par chat anonyme avec des discussions thématiques.
- **Espace psychologique** : Ressources et exercices pour gérer l’anxiété sociale et se détendre.
- **Suivi des niveaux** : Un système qui suit l’évolution des utilisateurs selon leurs missions complétées.
- **Mode nuit/jour** : Un design personnalisable selon les préférences visuelles des utilisateurs.
